# PerfEco Consulting — Site web

## Projet
Site vitrine bilingue (FR/EN) de PerfEco Consulting, cabinet de conseil en performance organisationnelle et transformation digitale, fondé par Jean-Michel Blaise. Cible : dirigeants (DG/DAF/DSI/COO) en Nouvelle-Calédonie et Polynésie française.

## Stack technique
- Astro 5.18 + TailwindCSS 3.4
- Site statique, pas de CMS backend
- Déploiement sur Netlify

## Commandes
- npm run dev : serveur de développement
- npm run build : build de production
- npm run preview : prévisualisation du build
